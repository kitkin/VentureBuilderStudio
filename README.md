# Venture Builder Studio

Рабочая папка по текущему чату: обработка идей маркетологов, выбор одной идеи и проработка воронок условно-бесплатного трафика.

## Содержимое

| Файл | Описание |
|------|----------|
| **Ideas_Report_YC_Pitch_and_Metrics.md** | Отчёт по 7 идеям **Надежды**: сводная таблица, YC High Concept Pitch, ключевые метрики; план выбора одной идеи и воронок бесплатного трафика |
| **Ideas_Report_Sebastian_YC_Pitch_and_Metrics.md** | Отчёт по 7 идеям **Себастьяна**: сводная таблица, YC High Concept Pitch, ключевые метрики, риски и реализуемость |
| **Ideas_Pool_Summary_14.md** | Сводный пул всех 14 идей (краткие one-liner) для сравнения и выбора |
| **All_14_Ideas_Simple.html** | Простой обзор всех 14 идей на английском: карточки с проблемой, решением, ценой, конкурентом, риском, ЦА (для быстрого просмотра) |
| **Projects_Dashboard.html** | Интерактивный дашборд: все 14 идей (аккордеоны), deep dive по проработанным (подтабы), план работ с фазами (чек-листы) |
| **Product_Brief_Review_AI_Nadezhda.md** | Бриф по Review AI (Надежда): ЦА, гео, каналы (Email, Reddit, Product Hunt, Quora), конверсии, триггеры, ожидаемые лиды |
| **Product_Brief_Influencer_AI_Sebastian.md** | Бриф по Influencer AI (Себастьян): рынок, продукт, кто покупает и когда, триггеры, возражения, конкуренты, бесплатный трафик, монетизация, MVP, моат |
| **Selected_Idea.md** | *(заполнить после выбора)* — какая идея выбрана и по каким критериям (из пула 14 идей) |
| **Funnel_Plan_[Название].md** | *(создать после выбора)* — детальный план воронок условно-бесплатного трафика для выбранной идеи |

## Источники

- **Надежда:** чек-листы самопроверки — суть идей, боль, решение, конкуренты, риски, цены, выгоды, каналы продвижения (7 идей).
- **Себастьян:** чек-листы самопроверки — суть, боль, рынок/конкуренты, transaction cost, реализуемость, соответствие навыкам (7 идей).

**Итого:** 14 идей в едином формате (YC pitch + метрики) для выбора одной и проработки воронок.
